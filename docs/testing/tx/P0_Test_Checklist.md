## 1. 目标

将当前阶段测试拆为两部分：

- `P0-A`：在纯 MATLAB 环境下完成参数级自洽检查；
- `P0-B`：在硬件到位后完成 USRP 基础链路验证。

只有 `P0-A` 通过后，才建议进入 `P0-B`。

---

## 2. 前置条件

### 2.1 P0-A 前置条件

- [ ] 已阅读 `AI_CONTEXT.md`
- [ ] 已阅读 `../../planning/BOC_TDMA_System_Planning.md` §6 阶段 A
- [ ] MATLAB 环境可打开并运行脚本

### 2.2 P0-B 前置条件

- [ ] `P0-A` 已通过
- [ ] USRP 已到位并可被主机识别
- [ ] 频谱仪、SMA 电缆、必要衰减器已到位
- [ ] 已阅读 `../../phases/tx/P0_USRP_Link_Verification.md`

## 3. P0-A：参数级自洽检查

### 3.1 执行步骤

| 编号 | 执行项 | 预期输出 | 状态 | 证据 |
|---|---|---|---|---|
| A-01 | 打开并阅读 `../../../matlab/check_params.m` | 确认脚本无外部输入、只做参数检查 | 未开始 | 阅读记录 |
| A-02 | 在 MATLAB 中运行 `check_params.m` | 脚本成功执行，打印检查结果 | 未开始 | MATLAB 控制台输出 |
| A-03 | 核对 `PASS / WARN / FAIL` 摘要 | `FAIL = 0` | 未开始 | 结果截图 |
| A-04 | 核对唯一允许的 `WARN` | 仅 `fs/fsc = 2.4` 被标为 `告警` | 未开始 | 结果截图 |
| A-05 | 核对关键派生参数 | `Tc`、`Nc`、`Nslot`、`Tslot`、`Nframe`、`Tframe`、数据率、处理增益均正确 | 未开始 | 控制台输出 |
| A-06 | 写出是否允许进入 `P0-B` 的结论 | 明确 `允许 / 不允许` | 未开始 | 执行记录 |

### 3.2 P0-A 验收标准

- `PASS` 项全部通过；
- `WARN = 1`，且唯一告警项为 `fs/fsc = 2.4`；
- `FAIL = 0`；
- 结论明确写明“当前仍未触碰硬件”。

### 3.3 P0-A 证据归档

- MATLAB 控制台输出截图
- `PASS / WARN / FAIL` 摘要记录
- 是否进入 `P0-B` 的结论

## 4. P0-B：USRP 基础链路验证

### 4.1 执行步骤

| 编号 | 执行项 | 预期输出 | 状态 | 证据 |
|---|---|---|---|---|
| B-01 | 运行设备探测命令（如 `uhd_find_devices` / `uhd_usrp_probe`） | USRP 被识别，型号 / 接口 / 固件信息可读 | 未开始 | 终端输出 |
| B-02 | 记录 USRP 型号、接口方式、UHD / 固件版本 | 基线参数表完整 | 未开始 | 参数记录表 |
| B-03 | 配置 36 MSPS 采样率与主时钟 | 实际生效采样率正确 | 未开始 | 配置截图 / 日志 |
| B-04 | 生成 1 MHz 左右复单音 CW IQ 数据 | 单音基带文件可导出 | 未开始 | MATLAB 输出 |
| B-05 | 检查 IQ 样本长度、归一化、sc16 格式 | 无 NaN / Inf，量化格式正确 | 未开始 | 自检记录 |
| B-06 | 发射 CW 并连接频谱仪观测 | 频谱仪上出现稳定单峰 | 未开始 | 频谱截图 |
| B-07 | 检查是否存在异常 DC 峰或明显镜像 | 基线可接受 | 未开始 | 频谱截图 |
| B-08 | 进行 2 分钟稳定性测试 | 无 underflow / 中断 | 未开始 | 终端日志 |
| B-09 | 保存截图、日志和参数记录 | 测试证据完整可追溯 | 未开始 | 文件清单 |
| B-10 | 写出是否允许进入 P1 / P2 的结论 | 明确 `允许 / 不允许` | 未开始 | 执行记录 |

### 4.2 P0-B 验收标准

- USRP 可被稳定识别；
- 36 MSPS 配置成功且记录了主时钟关系；
- 频谱仪上稳定看到预期单峰；
- 2 分钟稳定性测试无 underflow；
- 截图、日志、配置参数均已归档。

### 4.3 P0-B 证据归档

- UHD 设备探测输出
- 采样率 / 主时钟配置记录
- CW IQ 生成记录
- 频谱仪截图（含中心频率、RBW、参考电平）
- 2 分钟稳定性日志
