## 1. 目标

将当前阶段测试拆为两部分：

- `P1-A`：在纯 MATLAB 环境下完成 BOC(15,3) 子载波离散化分析、方案 A/B 对比和基线锁定；
- `P1-B`：在 `P0-B` 已完成后，按需补做 USRP + 频谱仪观测，确认硬件观测与 MATLAB 基线一致。

在当前项目状态下，默认优先执行 `P1-A`。`P1-B` 若暂不触碰硬件，可标记为 `阻塞`，但这不阻塞 `P1-A` 推进。
