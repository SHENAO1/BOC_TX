# P6 测试清单

**文档版本：** v1.0  
**文档状态：** 骨架

## 1. 目标

对关键发端参数进行扫描、量化和锁定，形成可供接收端使用的最终发端参数规格书。

---

## 2. 前置条件

- [ ] P5 已通过
- [ ] 稳定发射基线已锁定
- [ ] 已阅读源阶段文档 §3、§4、§10、§12

---

## 3. 最少执行项

| 编号 | 执行项 | 预期输出 | 状态 | 证据 |
|---|---|---|---|---|
| T-01 | 建立统一参数扫描对象清单 | A/B/C 类参数清楚 | 未开始 | 参数表 |
| T-02 | 建立统一评价指标体系 | 频域 / 相关域 / 工程实现指标明确 | 未开始 | 指标表 |
| T-03 | 扫描发射增益与关键信号生成参数 | 影响可量化 | 未开始 | 对比图 / 表 |
| T-04 | 对比 `sign(0)` 方案和截短策略 | 推荐值清楚 | 未开始 | 对比报告 |
| T-05 | 形成最终参数锁定表与规格书 | 可供 P7 / 接收端引用 | 未开始 | 参数规格书 |

---

## 4. 验收标准

- 基线参数可被扫描框架复现；
- 所有推荐参数有明确结论；
- 若推荐参数改变了 P5 基线，应明确是否需要补做 P5 复验；
- 以源阶段文档 §12 为最终准入门槛。
